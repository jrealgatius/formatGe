# FormatGe

FormatGe contain functions to Format and Generate variables to data table through structural external file (Conductor). Mainly the functions will to allow to clean, to debug, and to generate others variables  (Format and Generate), as the first step when a data table is receipt before to start the data analysis. 

This R package that to allow prepare a data table through external file, that contain information parameterized like labels, cut points, valid range values from each variable.

The functions make actions according the information that include this file. This method minimize mistakes due that this file can be reviewed and manipulated for non-R users and external team.
